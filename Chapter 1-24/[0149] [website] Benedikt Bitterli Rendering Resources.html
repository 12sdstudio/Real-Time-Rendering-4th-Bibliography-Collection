Rendering Resources

This page offers 32 different 3D scenes that you can use for free in your rendering research, publications and classes. They range in complexity from small test setups all the way to complex interior scenes with difficult indirect lighting. A number of hair models are also included. All scenes come with explicit licenses attached and have few restrictions: The majority allow commercial use, and many don't even require attribution.

Most of these scenes were generously made available by artists on Blendswap.com. I picked a few of these scenes, cleaned up the geometry, exported it to OBJ and then manually set up PBR materials, lighting and camera. This process is time intensive (up to several days per scene), and I am releasing what I've done in the hopes that it will be useful to others. The scenes that I authored myself are released into the public domain.

All scenes were set up in Tungsten initially and exported to Mitsuba and pbrt-v3 using automatic converters. Since Mitsuba and pbrt-v3 don't fully support the tonemaps and BRDFs used here, conversion is approximate and Mitsuba/pbrt-v3 output may deviate from the renderings shown here. The Tungsten binaries that produced these renderings are available on Github.

Special thanks go to Alex Headley and Nicolas Frattaroli for providing the hardware to render the images on this page.

How to cite

If you use these scenes in a publication, I would appreciate (but don't require) a citation using the following BibTeX template:

@misc{resources16,
   Author = {Benedikt Bitterli},
   Year = {2016},
   Note = {https://benedikt-bitterli.me/resources/},
   Title = {Rendering resources}
}

If you want to contribute a scene, leave a comment or just want to say hello, feel free to write me an email.
